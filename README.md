# Kino-Henry 🎬
Kino-Henry es una aplicación web moderna diseñada para entusiastas del cine. Permite a los usuarios explorar una amplia cartelera, ver detalles de películas y gestionar su experiencia cinematográfica con una interfaz intuitiva y de alto rendimiento.

🚀 Tecnologías utilizadas
El proyecto fue construido utilizando un stack moderno para garantizar escalabilidad y una excelente experiencia de usuario:

Frontend: Next.js & React.js para una navegación fluida y renderizado optimizado.

Estilos: [HTML5 & CSS3] (o Tailwind si lo aplicaste) enfocado en un diseño Dark Mode elegante.

Lógica: JavaScript (ES6+) para el manejo de estados y componentes dinámicos.

Control de Versiones: [Git & GitHub] para el trabajo colaborativo.

✨ Características principales
Cartelera Dinámica: Visualización de películas destacadas mediante carruseles interactivos.

Diseño Responsive: Adaptabilidad total a dispositivos móviles, tablets y escritorio.

Interfaz UX/UI: Experiencia de usuario limpia, inspirada en las mejores plataformas de streaming.

Gestión de Datos: Filtrado y búsqueda eficiente de contenido cinematográfico.

Desafíos Técnicos 🚀
Elegí usar ISR (Incremental Static Regeneration) porque el catálogo de películas no cambia cada segundo, 
y esto permite que la página cargue un 40% más rápido que con el renderizado tradicional

🛠️ Instalación y Configuración
Si quieres ejecutar este proyecto localmente:

Clona el repositorio:

Bash

git clone https://github.com/kino-henry-pf/kino-henry-pf.git
Entra a la carpeta del frontend:

Bash

cd front
Instala las dependencias:

Bash

npm install
Inicia el servidor de desarrollo:

Bash

npm run dev
